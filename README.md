# SyncSettings
A Visual Studio Code extension to synchronize your settings.

## Process

### Backup

1. Get all Settings (settings, keybindings, snippets, Extension list)
2. Backup all settings

### Restore

1. Get all Settings (settings, keybindings, snippets, Extension list)
2. Get Backup
3. Restore

### Merge

1. Get all Settings (settings, keybindings, snippets, Extension list)
2. Take local | Take backup | ask every file | ask every entry
